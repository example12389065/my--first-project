My Mini Search Engine Project

What I Built

I created a simple search engine website. Users type a word like youtube or github. The website redirects them to that website.

How It Works

The website reads what the user types. Then it checks the word. If it matches something known, it opens that website automatically.

Main Code Explained

function searchSite(){ let text = document.getElementById("search").value; }

This line reads the text written inside the search box.

Decision Logic

if(text === "youtube"){ window.location="https://youtube.com"; }

If user types youtube, the browser moves to youtube website. window.location changes the page.
