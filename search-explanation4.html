How I Built My Mini Search Engine

1. Idea

I wanted users to type a word and go directly to a website. Example: youtube → opens YouTube. github → opens GitHub.

2. Full Search Engine Code

<!DOCTYPE html> <html> <head> <title>Search Page</title> <script> function searchSite(){ let text = document.getElementById("search").value.toLowerCase(); if(text === "youtube"){ window.location="https://youtube.com"; } else if(text === "github"){ window.location="https://github.com"; } else{ alert("Not Found"); } } </script> </head> <body> <h1>Mini Search Engine</h1> <input id="search" placeholder="type youtube or github"> <button onclick="searchSite()"> Search </button> </body> </html>

3. Line By Line Explanation

<!DOCTYPE html> → tells browser modern HTML.

<html> → container of entire webpage.

<head> → information about website.

<script> → JavaScript logic starts.

function searchSite() → instructions that run when button clicked.

document.getElementById → finds search box.

.value → reads user typed word.

.toLowerCase() → converts text to small letters.

if statement → checks condition.

window.location → moves browser to another website.

alert → popup message if not found.

4. What I Learned

  • HTML builds structure.
  • JavaScript controls behaviour.
  • Decision logic changes navigation.
  • Websites respond to user input.
